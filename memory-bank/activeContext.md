# Active Context

## Joriy Fokus
Hozirda asosiy e'tibor loyiha uchun "Memory Bank" tizimini to'liq joriy etishga qaratilgan. Bu loyihani boshqarishni osonlashtiradi va keyingi sessiyalarda kontekstni yo'qotmaslikka yordam beradi.

## So'nggi O'zgarishlar
-   `GEMINI.md` dagi ko'rsatmalar asosida Memory Bank strukturasi yaratildi.
-   `projectbrief.md`: Loyiha maqsadlari belgilandi.
-   `productContext.md`: Muammo va yechimlar tahlil qilindi.
-   `systemPatterns.md`: Tizim arxitekturasi chizildi.
-   `techContext.md`: Texnologik stack hujjatlashtirildi.

## Keyingi Qadamlar
1.  Memory Bank fayllarini to'ldirishni yakunlash.
2.  Litsenziya tizimini olib tashlash bo'yicha ishlarni rejalashtirish (avvalgi suhbatlardan kelib chiqqan holda).
3.  Admin panelda restoranlarni boshqarish funksiyasini tekshirish.

## Faol Qarorlar
-   Barcha hujjatlar **O'zbek tilida** yuritiladi.
-   Documentation (.md) fayllar `memory-bank` papkasida saqlanadi.
